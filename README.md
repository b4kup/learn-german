# learn-german

Two interactive German exercises for Danish learners (A2–B2): a vocab/grammar/false-friends trainer and a "day in Berlin" story-mode lesson. Dependency-free static HTML — no build, no framework.

## Run locally

```bash
python3 -m http.server
```

Open http://localhost:8000.

## Live

https://kiki-learn.justusernst.de

## Files

- `index.html` — landing hub
- `a2-b2-trainer.html` — vocab + grammar quiz
- `tag-in-berlin.html` — Berlin story lesson
